# 工具安裝指南

## Binwalk

```bash
# Ubuntu/Debian
sudo apt-get install binwalk

# macOS
brew install binwalk
```

## Ghidra

1. 從 [Ghidra 官方網站](https://ghidra-sre.org/) 下載最新版本
2. 解壓縮下載的檔案
3. 執行 `ghidraRun` 腳本

## YARA

```bash
# Ubuntu/Debian
sudo apt-get install yara

# macOS
brew install yara
```

## python-can

```bash
pip install python-can
```

## 其他依賴

```bash
pip install pyyaml
```
